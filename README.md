## Dry Fruits Ordering System

## About the Project

This is a simple Python-based project that helps users to  order dry fruits easily from the shop.
This system allows customers to view products, select items, choose quantities, and generate a final bill automatically.
This project is inspired by a real-life problem, my family owns a dry fruit shop.My goal was to create a small project but effective system for manual billing.

## Features

* Display menu of dry fruits with prices
* Select items using number input
* Choose quantity (250g, 500g, 750g, 1kg or random)
* Add items to the cart
* Generates final bill with total price
* Order confirmimg system
* Error for invalid inputs
